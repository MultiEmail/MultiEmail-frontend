Setup

Windows

 
git clone https://github.com/MultiEmail/MultiEmail-frontend.git
cd MultiEmail-frontend
yarn install
yarn dev

Linux

 
git clone https://github.com/MultiEmail/MultiEmail-frontend.git && cd MultiEmail-frontend && yarn install && yarn dev
